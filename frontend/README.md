#  Collide
### Назначения файлов
- assets: Все изображения, иконки, svg, шрифты 
- components: Переиспользуемые UI-компоненты (кнопки, карточки, формы и т.д.)
- features: Большие функциональные блоки (по смыслу)
- hooks: Все кастомные React хуки 
- pages: Отдельные страницы приложения 
- services: Всё, что связано с запросами к серверу 
- utils: Вспомогательные чистые функции
